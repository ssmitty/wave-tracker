Portfolio project

Wave Vision

A Streamlit and OpenCV demo for surf video analysis. Upload a fixed-camera beach clip, calibrate the scene with draggable overlays, and estimate wave height, velocity, and surf conditions.

View code GitHub repo
What it demonstrates
  • Computer vision with classical OpenCV
  • Draggable scene calibration
  • Weather and wind context from Open-Meteo
  • Generated surf reports grounded in metrics

Core capabilities

The app estimates wave count, relative height, average velocity, and confidence from uploaded surf footage. It also pulls wind and weather data for a location you enter.

Built for a resume

This is intentionally scoped as a polished demo, not a production system. The goal is to look sharp, be explainable, and show real engineering tradeoffs.

Tech stack

Python, Streamlit, OpenCV, Open-Meteo, and optional OpenAI report generation.
